# ⛏️ HNS Mining Pool Accountability Tracker

> A community-maintained transparency record of Handshake (HNS) mining pool behavior — who stayed, who left, and when.

---

## 🎯 Purpose

When HNS price drops, some mining pools quietly exit — abandoning the network and their miners without notice. This repository keeps a **permanent public record** of pool behavior during difficult market periods so that:

- **Miners** can choose pools with a proven track record of loyalty
- **The community** can reward pools that support the network long-term
- **New miners** entering HNS have transparent history to reference

This is not a place for personal attacks. It is a factual, evidence-based ledger of pool decisions.

## 🏆 Loyalty Score

Pools are rated on a simple community score:

| Score | Meaning |
|-------|---------|
| ⭐⭐⭐⭐⭐ | Never exited, consistent hashrate through bear markets |
| ⭐⭐⭐⭐ | Minor disruptions, communicated clearly with miners |
| ⭐⭐⭐ | Temporarily reduced support, returned afterward |
| ⭐⭐ | Exited without notice, returned later |
| ⭐ | Exited during downturn, never returned |

## 📣 How to Submit a Report

1. **Open an Issue** using the [Pool Event Report](https://github.com/HellotoSinaa/hns-pool-accountability/blob/main/pool-report.md) template
2. Include verifiable evidence (announcements, hashrate charts, dates)
3. A maintainer will review and add it to the appropriate pool file
4. Pull Requests with evidence are also welcome

> ⚠️ **No speculation or rumors.** Every entry must have a verifiable source.
